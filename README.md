# Chang-Syu Chen — Portfolio

A modern, dark-themed portfolio site built from CV content.

## Run locally

```bash
# From this directory
open index.html
# or
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Deploy

- **GitHub Pages**: Push to a repo and enable Pages on the `main` branch (root or `/docs`).
- **Vercel / Netlify**: Drag the `portfolio` folder or connect the repo.

All assets are local; no build step required.
